# Student Performance Predictor

## Description
This project predicts student marks based on study hours, sleep hours, and attendance using a machine learning model.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## How to Run
1. Open the Jupyter Notebook file
2. Run all cells
3. Enter custom input values to predict marks

## Features
- Predicts student performance
- Uses Linear Regression
- Visualizes data using graphs

## Sample Input
Study Hours: 6  
Sleep Hours: 7  
Attendance: 85  

## Output
Predicted Marks: ~76
